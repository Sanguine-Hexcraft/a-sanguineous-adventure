# a sanguineous adventure

A dark-fantasy EverQuest fan site — character journal, lore codex, and achievement tracker for the MMORPG EverQuest.

> *Track the deed. Preserve the story. Understand the world.*

**Live site:** https://a-sanguineous-adventure.vercel.app

---

## What this is

A personal project for tracking an EverQuest character's journey through Norrath. The site feels like an adventurer's codex — not a spreadsheet or a wiki. Progress reads like legend entries. Achievements are "sealed in the Codex." Notes are written "in the Chronicle."

Four pillars:
1. **Journey tracking** — zones, kills, quests, achievements, factions, notes, screenshots
2. **Achievement system** — all 5,944 retail EQ achievements imported, browsable by expansion and category
3. **Lore codex** — zone pages, NPC pages, faction pages, expansion lore in Markdown
4. **Log parser** — read EQ log files to detect events and update progress automatically

## Achievement data pipeline

Raw EQ client files → normalizer → browser-ready JSON.

To re-run the normalizer (e.g. if raw files are updated):
```bash
node scripts/normalize-achievements.js
```

Output goes to `public/data/achievements/`. The raw source files in `data/AchievementAssociationsClient-raw/` are never modified.

**Category structure:** blank parentId = top-level (expansion or group), numbered parentId = subcategory. Top-level IDs map directly to expansions: 100 = EverQuest (Classic), 200 = Ruins of Kunark, 300 = Scars of Velious, etc. through 3100 (latest). Special groups: 10 = General, 50 = Tradeskill, 70 = Hero's Journey, 80 = Slayer, 90 = Events.

## Routes

**Public**
- `/` → redirects to `/app`
- `/codex` — lore archive index
- `/codex/zones`, `/codex/npcs`, `/codex/factions`, `/codex/expansions` — category listings
- `/codex/[...slug]` — individual lore entries (powered by Nuxt Content)
- `/achievements` — full achievement browser

**App (will be auth-protected in Phase 3)**
- `/app` — character dashboard (chronicle)
- `/app/logs` — log parser
- `/app/notes` — lore notes
- `/app/media` — media gallery
